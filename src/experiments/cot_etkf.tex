The trained neural network inside the ensemble Kalman filter is tested as a final validation for the model.

Evaluating the neural network produced linear warping operator in the ensemble Kalman filter method presents a more intricate task.
Due to the computationally involved ETKF algorithm, a quick computation of the error metrics for a larger dataset is not feasible as with the deterministic model.
As the main focus of this work was on integrating the neural network as a generator of the state model operator, the observation model was simulated with the same training data with missing values to test the concept.
However, as discussed in Subsection~\ref{subsec:cotpreprocessing}, the missing values do present a significant problem with many real data sources.
Hence, the exercise is not entirely unrealistic even though the end objective would be some form of more sophisticated inverse problems such as data fusion. 
Due to the impossibility of knowing the actual complete state of the uninterpolated data, the visual examples are constructed by synthetically removing data values to simulate partial observations. 
Also, due to the separate problems of ensemble generation reflecting a realistic uncertainty for pure prediction problems, as discussed earlier, and validation metrics related to probabilistic ensemble forecasts, the forecasting problem tests are not directly presented.

Similar COT images outside of the training dataset of $128 \times 128$ pixels and similar synthetic examples are used as with evaluations of the deterministic model.
Despite using images of the same shape as the neural network works with, it is not a hard limitation of the proposed framework.
Larger images could be downsampled for the neural network model to create a state operator to work with larger state vectors in the ETKF or vice versa.
In order to visualise the performance of the motion models, the presented example images are selected such that they largely adhere to the advection equation with some random variation to demonstrate observation assimilation.

For fine-tuning the ETKF model integrating the deterministic neural network presented in Subsection~\ref{subsec:etkfmodel}, a few assumptions were made related to the chosen objectives of the verifications with the COT data and synthetic examples.
The ETKF parameters were chosen to give a small uncertainty for the observations, intending to return the observation values where available and, otherwise, values warped from the previous state.
This choice stems from the quickly evolving COT data not only within the model's capability but also to verify the results better.
In possible operational use of the framework, these parameters are naturally subject to case-by-case tuning depending on the actual data.
Particularly, the observation uncertainty standard deviation should be a realistic uncertainty estimate of the measurements instead of the assumed constant of 0.23 used for these examples.
For best results, a similar scaling as when training the network of $[0, 1]$ range of the data was also used for the ETKF\@.
As the perturbations for the ensemble according to the strategy presented in Subsection~\ref{subsubsec:etkftuning} result in a slightly different data range for the state vectors, the neural network scales the data before generating updated warping operators.
For the adding spatially varying noise in the generation of the initial ensemble and inflation, noise is generated for the entire image area but multiplied by the image values pointwise before adding to the images.
Localisation is performed in $16 \times 16$ pixel spatial blocks to maintain spatial structures in the data better.
An ensemble size of 500 is used as it appears to give a good balance of high-quality results and adequate computing time.

Furthermore, an incoming observed square appears in the filtered images as expected.
Thus, this example affirms that the method works as intended.

While Kalman filtering has the benefit of offering uncertainty estimates, a significant difficulty in these kinds of tasks is that the filter parameters effectively control the magnitude of uncertainty.
Especially with the ensemble filters where the used inflation technique essentially defines the state propagator model uncertainty, many unknown parameters must be guessed or estimated by hand.
While accurate inflation methods would be crucial in order to get accurate uncertainty estimates, without much data to do verification with, the crucial parameters must often be tuned with just the goal of obtaining visually pleasing results.
In wider use, however, the model uncertainty could be better estimated from previous model errors to get more accurate information.

The inflation and initial ensemble generation are even more critical in possible forecast usage as no observations are available to support the forecasts.
In addition, as the computation of a forecast distribution in the ensemble filter is not limited by the standard Kalman filter assumptions, the range of possibilities for ensemble generation and inflation is much broader.
In a recent work related to the forecasting problem presented in~\cite{intraday}, an ensemble is generated by multiplying the Fourier transforms of a noise field and a state estimate.
In that work, an advection model is also combined with an autoregressive model that could potentially improve the predictive performance of the neural network of this work concerning the COT data.
