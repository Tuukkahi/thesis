In the following subsection, the ensemble based Kalman filter set up of the later experiments is described.

The neural network is trained to produce advection fields from spatio-temporal advective data but is only able to work with complete data and can only produce deterministic forecasts.
As real-world observations of any process are rarely complete, or without uncertainties, the neural network is used in the Bayesian filtering context to treat spatio-temporal data in the state-space model.
The motivation of implementing this method is twofold.
Firstly, approximating the filtering distribution provides a way to assimilate observations that may come from multiple sources with varying quality to the state estimates as they come.
Secondly, approximating the prediction distribution yields probabilistic forecasts over multiple time steps.
While the main focus of this work has been on the filtering distribution, the prediction distribution case is also discussed as it comes with some intricacies of its own that need to be addressed.
A third closely related problem but not considered in the scope of this work would be estimating the smoothing distribution with a Kalman smoother to update the previous state estimates given new data.

Due to the high dimensionality of image data, the ensemble transform Kalman filter presented in Algorithm~\ref{alg:etkf} is used and therefore the distributions are only approximations instead of the analytical solutions of the standard Kalman filter.
The concept of using the neural network model as a state evolution operator in an EnKF takes influence from~\cite{deepide} but here a slightly different approach is taken to accomodate for even larger state vectors and as opposed to the interest in probabilstic forecasting, here the main interest is in the filtering distribution of filling out missing values from observations.

\subsubsection{State-Space Model Operators}
For analysis of the state-space model~\eqref{eq:statespacemodel}, observation and state evolution operators $H$ and $M$ are needed.
While the nonstationary inverse problem related to the observation operator make the Kalman filter as powerful method as it is, the main contribution and interest of this work is in the incorporation of the neural network in the state evolution operator.
Therefore, to separate the questions related to data fusion and more complex problems in order to simplify the later validation of this work, the observation operator is assumed to be a trivial identity mapping but leaving out missing observations from the state vectors.
But even with this treatment of the observation process, valuable problems related to missing observations that pose a difficult challenge in many applications may be approached.

As the neural network models data through the advection equation by generating advection fields, the output of the network may be interpreted as the linear warping function that transforms a state vector to the next time step.
Consequently, the state operator can be then defined to be the warping operator that only transforms the coordinate space of the mean state vectors, i.e.
\begin{equation*}
    M_{k+1}x_k[i,j] = 
        x_{k}[i-\mathbf{F}_{u}[i,j], \, j - \mathbf{F}_{v}[i,j]],
\end{equation*}
when the $i-\mathbf{F}_{u}[i,j]$ and $j - \mathbf{F}_{v}[i,j]$ are inside the image domain.
From outside the image domain, the border values must be taken in order to fill the entire image vector.
While this makes the estimation of border values problematic, the Kalman filter methods correct the states by observations when available as long as the observations can be trusted.
Moreover, the filtering can be performed over a larger domain than necessary, and focus on the interior of the filtered states.
Figure~\ref{fig:warping} depicts how the warping operator works for a small $5 \times 5$ image.

Due to the use of the ensemble filter, the same model operator is applied for each of the ensemble members separately according to the Algorithm~\ref{alg:etkf}.
As the neural network is intended to produce advection fields from single images, the ensemble means are used to update the operator when needed.
Here a distinction is needed for the filtering and prediction distributions.
When computing the filtering distributions, incoming observations give new information on the state of the system and therefore an updated model operator is computed to give the most up to date advection field estimate.
However, as observations are not available for the prediction case, the best guess for the model operator is the one computed from the latest filtering distribution means.

\subsubsection{ETKF settings\label{subsubsec:etkftuning}}
While the ETKF model in Algorithm~\ref{alg:etkf} provides a computationally efficient for Kalman filter estimates, some tuning and parameters are needed for optimal results.
Especially in the case of advective spatio-temporal data considered in this work, some tuning is needed to get realistic state estimates.

In using ensemble Kalman filter methods, an initial ensemble of the state vector is needed before running the algorithm, and in the case of integrating the ETKF with the proposed neural network, three latest complete state vectors are needed to obtain the first advective model operator. 
The ensemble size is an important parameter that represents a trade-off between accurately representing the true theoretical state distribution and computational efficiency with larger state vectors.
Even though the initial ensemble of the state vecor must also be computed without the help of the Kalman filter, the assimilation of observations in subsequent steps corrects also the first estimate over time.
While many data sources may have some theoretical uncertainty estimates, often the ensemble must be created by perturbing only a single initial state vector.
While Kalman filters provide an efficient way optimally assimilate observations to the state estimates but a consquent assumption is that the uncertainties are additive and Gaussian which in the ensemble case means state the ensemble should follow a Gaussian distribution.
The assumptions are needed for the efficient formulations but the suitability of the assumptions to the used data may wary.

This is also an important consideration when computing the filtering and prediction distributions separately as the assumptions are needed only for optimality of the observation update step.
As the prediction distribution case with the ETKF works essentially by propagating the latest state ensemble estimate forward, the ensemble need not to be limited by the Gaussian distribtion assumptions.
Thus, for the prediction distribution case the ensemble can be generated and inflated on a case by case basis depending on the exact data source used and e.g.\ on the quality of previous predictions.

But also for the filtering distribution case, a non-religious following of the Gaussian assumptions is necessary.
Given the advective data considered here, the uncertainties are known to be related to location of spatial objects in data and their values.
Therefore, while not theoretically entirely consistent with the Kalman filter assumptions, the initial ensemble is created by applying Gaussian noise of a small intensity, and applying uniform random warps to the first state estimate.

As the same model operator given by the neural network is applied to all members of the ensemble, the borders of the state become largely similar between ensemble members.
As the ensemble should represent the uncertainty in the state, this easily underestimates uncertainty in the border areas.
To address this, inflation techniques are needed in increasing the variation between ensemble members.
And for this, the similar techniques as with creating the initial ensemble are used, i.e.\ perturbing the ensemble by Gaussian noise and applying small uniform warping operations to each member of the ensembles.

Another issue rising from the nature of advective spatio-temporal data is that the spatial structures are easily broken down by small ensembles and the assimilation of large observations.
Therefore, localisation is performed by dividing the state to smaller patches where the ETKF algorithm is performed separately.
As the trivial observation operator is not considering values outside the domain, the localisation behaves well on the borders of the patches.
Another benefit of localisation is the possibility of speeding up computation significantly by performing the algorithm in parallel for each of the smaller blocks.
