In this subsection the neural network model predicting advection fields from two-dimensional spatio-temporal data is described.

The data the network is designed to work with is assumed to follow the discrete advection equation in two spatial dimensions of same length:
\begin{equation*}
    \frac{\partial I_t(x,y)}{\partial t} = -\mathbf{F}_{u}(x,y) \frac{\partial I_{t}(x,y)}{\partial x} -\mathbf{F}_{v}(x,y) \frac{\partial I_{t}(x,y)}{\partial y}, \quad I \in \R^{n \times n}, \, t = 0,1,\dots.
\end{equation*}
Specifically, the data consists of some smooth objects moving uniformly across the two spatial dimensions over time.
Using a sequence of this data, the network predicts an advection field that represents the advection process across the sequence.
Even in the presence of noisy data, the objective is to produce smooth advection fields.
While the network is trained and evaluated mainly on satellite images of meteorological data, it is intended to be applicable to all similar advective processes.

Advection field estimation has also been studied in the context of optical flow problem.
Classical optical flow techniques are also often used in meteorology in nowcasting problems~\cite{pulkkinenetal}.
The neural network approach has also been studied but mostly focusing on different objectives.
They are often trained with ground truth flows and with datasets where motion only applies to small objects in images most notably with the KITTI~\cite{kitti}, and Sintel~\cite{sintel} datasets.
Contrary to this, the model used for this thesis has to be trained in an unsupervised fashion as the ground truth advection fields are not available, and the data for which the model is intended to be used with is very different from the usual ones used in the neural network optical flow context.

A similar unsupervised approach, from where much of the inspiration to the main principles of the neural network model can be attributed, to is explored in~\cite{debezenac} and~\cite{deepide}.
In these works an advection field neural network was used with much more complex sea surface temperature data with the attempt to generate pixel-wise advection vector estimates.
Moreover, these works also modelled the diffusion parameter,~\cite{debezenac} by a hardcoded value and~\cite{deepide} by a separate neural network.
The choice of leaving out diffusion from this model primarily stems from the fact that the cloud data with which the model was developed does not exhibit diffusion and thus lack physical interpretation.
Also in using a neural network for diffusion modelling specific care must be given to the loss function selection as diffusion is easily overestimated if minimising average prediction errors.
Nevertheless, for incorporating a diffusion parameter to this model, both methods could be added without changing the advective model. 

In order to tackle the needs of the much lower-dimensional data similar to the one described in Section~\ref{subsubsection:dynamicmodellingspatiotemporaldata} and in Figure~\ref{fig:advection_diffusion} but without diffusion, a different approach to the neural network structure is required.
As a consequent, the underlying neural network architecture is also designed differently.
Basis vectors are utilised here akin to the method used in~\cite{deepide}, but with two key differences.
A simple second degree polynomial is opted as opposed to a high-dimensional radial basis for the advection field which guarantees smooth advection fields.
Furthermore, instead of predicting the basis vector coefficients directly, the neural network predicts advection at fixed nodes from which a polynomial can be interpolated throughout the domain.
This chosen approach is better suited especially for the high resolution meteorological applications in mind as small-scale differences in the data behave mostly randomly, and the main features of the data mostly advect uniformly according to winds.

\subsubsection{Model Architecture}
The proposed model architecture is similar to the well-known Residual Networks (ResNet) that are deep yet computationally efficient.
To create two-dimensional advection vectors, two separate ResNet models are employed, each trained to predict a single component of the two-dimensional advection field: one for the u-component, and the other for the v-component.

Residual Networks are characterised by the use of skip connections, which allow the gradient to be directly backpropagated to earlier layers.
The ‘residual’ term comes from the learning process of these networks where they learn the residual mappings relative to the layer inputs, alleviating the common vanishing gradient problem in deep neural networks.
While residual connections are utilised in a wide array of neural network architectures, they were first formally developed for image recognition tasks in~\cite{resnet}.
The task of learning advection parameters at fixed nodes in this work bears resemblance to the image recognition tasks.
However, instead of outputting classes from multi-channel images, in this thesis, the network is interpreted as outputting advection field values at nodes from spatio-temporal data.

A key component of the ResNet architecture is the residual block.
Residual blocks have the general form
\begin{equation}\label{eq:residualblock}
    F(x) + x,
\end{equation}
where $F(x)$ is the residual mapping to be learned representing layers of the neural network. $x$ is the original input to the block which represents the skip connection.
The important aspect here is the behaviour of the residual blocks in backpropagation.
Deep neural networks suffer from a problem called diminishing gradients when the chain rule results in a large number of multiplication of small numbers and thus producing small gradients.
However, the derivative of the residual block
\begin{equation}\label{eq:residualblockderivative}
    \frac{d}{dx} F(x) + x = \frac{d}{dx} F(x) + 1,
\end{equation}
ensures that even with propagating gradients through a deep network, they don't vanish, thanks to the $+1$ term.

This kind of technique is applied in a wide array of neural network architectures, but perhaps the most common use is in convolutional neural networks.
In convolutional networks, the residual blocks consist of several convolution layers, each with batch normalisations and activations.
In this thesis, a 26-layer convolutional ResNet was deployed.

\subsubsection{Producing Advection Fields}
While the described architecture generates the low-dimensional vector of advection field values at only a few points, the primary objective is to produce the complete advection fields.
To this end, the basis vectors are constructed as polynomials across the spatial data domain using Lagrange polynomial interpolation.
The method of prediicting advection values from which the polynomial coefficients are then computed gives stability to optimising the network as changes to the output have a predictable change in the loss function value.

Second-degree polynomials are chosen as a strike between capturing a majority of the underlying advection process and maintaining sufficient flexibility to span the same advection field over multiple time steps.
Thus the advection fields are now of the form
\begin{equation*}
    \mathbf{F}(x,y) = 
    \begin{bmatrix}
        \mathbf{F}_{u}(x,y)\\
        \mathbf{F}_{v}(x,y)\\
    \end{bmatrix} =
    \begin{bmatrix}
        a_{u_0} + a_{u_1} x + a_{u_2} y + a_{u_3} x^2 + a_{u_4} xy + a_{u_5} y^2\\
        a_{v_0} + a_{v_1} x + a_{v_2} y + a_{v_3} x^2 + a_{v_4} xy + a_{v_5} y^2\\
    \end{bmatrix}.
\end{equation*}

Employing two separate ResNet models configured to output at six different nodes $\{ (x_i, y_i) \}_{i=0}^{5}$ the advection field polynomial values, a Vandermonde matrix can be constructed at the same set nodes as.
\begin{equation*}
    V((x_0,y_0),(x_1,y_1),\dots,(x_{5}, y_{5})) =
    \begin{bmatrix}
        1 & x_0 & y_0 & x_{0}^2 & x_{0}y_{0} & y_{0}^2\\
        1 & x_1 & y_1 & x_{1}^2 & x_{1}y_{1} & y_{1}^2\\
        \vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
        1 & x_{5} & y_{5} & x_{5}^2 & x_{5}y_{5} & y_{5}^2
    \end{bmatrix}.
\end{equation*}
Given this, the polynomial coefficients $a_u$ can now be solved uniquely by inverting the Vandermonde polynomial:
\begin{align*}
        \begin{bmatrix}
            1 & x_0 & y_0 & x_{0}^2 & x_{0}y_{0} & y_{0}^2\\
            1 & x_1 & y_1 & x_{1}^2 & x_{1}y_{1} & y_{1}^2\\
            \vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
            1 & x_{5} & y_{5} & x_{5}^2 & x_{5}y_{5} & y_{5}^2
        \end{bmatrix}
        \begin{bmatrix}
            a_{u_0}\\
            a_{u_1}\\
            \vdots \\
            a_{u_5}\\
        \end{bmatrix} &= 
        \begin{bmatrix}
            \mathbf{F}_{u}(x_0,y_0) \\
            \mathbf{F}_{u}(x_1,y_1) \\
            \vdots \\
            \mathbf{F}_{u}(x_5,y_5)
        \end{bmatrix}\\
        \begin{bmatrix}
            a_{u_0}\\
            a_{u_1}\\
            \vdots \\
            a_{u_5}\\
        \end{bmatrix} &=
        \begin{bmatrix}
            1 & x_0 & y_0 & x_{0}^2 & x_{0}y_{0} & y_{0}^2\\
            1 & x_1 & y_1 & x_{1}^2 & x_{1}y_{1} & y_{1}^2\\
            \vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
            1 & x_{5} & y_{5} & x_{5}^2 & x_{5}y_{5} & y_{5}^2
        \end{bmatrix}^{-1}
        \begin{bmatrix}
            \mathbf{F}_{u}(x_0,y_0) \\
            \mathbf{F}_{u}(x_1,y_1) \\
            \vdots \\
            \mathbf{F}_{u}(x_5,y_5)
        \end{bmatrix},
\end{align*}
and similarly for $a_v$.
As the Vandermonde polynomial is notorius for having a large condition number, the nodes need to be selected carefully to ensure stable invertibility.
Hence, the image domain is set to be $[-1, 1] \times [-1,1]$, and the first node set at the origin and five nodes at non-symmetric points on the unit circle as depicted in figure~\ref{fig:testpoints}.
\begin{figure}[h]
    \centering
    \input{model/fig/testpoint.pypgf}
    \caption{\label{fig:testpoints}The six nodes at which the neural network approximated advection field values. In order to ensure stable inversibility of the two-dimensional Vandermonde matrix, the location of the nodes must be selected carefully.}
\end{figure}
After solving the polynomial coefficients, the advection vectors may then be interpolated over the entire domain by evaluating the Vandermonde matrix at every pixel and right-multiplying it by the coefficient vectors.  

Since the basis vectors are constant, the model complexity is effectively controlled by the dimensionality of the basis.
Consequently, the selection of the basis represents a trade-off between smoothness of the advection fields and the ability to fit more complex patterns in the advection fields.

\subsubsection{Training the Neural Network}
An unsupervised training approach is employed due to the general unavailability of ground-truth advection fields.
The loss computation in training has to be thus done trough warping the latest data sample according to the predicted advection field, and comparing it to the true data samples of corresponding time steps.

The warping operation essentially transforms the data coordinates in alighment with the predicted advection fields.
During training phase, bilinear interpolation is applied to execute this operation.
This method calculates the value at a new location in the warped data as a weighted average of the values from the nearest four locations in the original data, ensuring differentiability for backpropagation.
However, this method also tends to smooth out the warped images.
Consequently, a simpler nearest-point interpolation method is used in inference.

The network is trained by minimising the average MSE loss between warped images, and the target images over a predetermined number of time steps.
For each iteration, the network predicts the advection fields from a batch of $\tau$ consecutive data samples, and the latest image from this batch is warped using the aforementioned warping scheme.
The warping is done over several subsequent time steps to compute an average loss between actual images, and the warped images.
Due to the lack of exterior information beyond the image boundaries, the model cannot predict the edge values.
Therefore, the edge values are discarded when computing the MSE loss.
The selection of the $\tau$ and the number of time steps to warp in training depend on the temporal characteristics of the training data.

As the optimisation algorithm, Adam is employed in optimising the model.
It is a variant of the already discussed SGD method but due to its adaptive step size it is often more efficient than the SGD.
